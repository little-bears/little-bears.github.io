React 属于单页面应用（SPA），相对于传统多页面应用不同页面跳转形式，单页面应用只有一个 Web 页面，通过隐藏显示对应的页面片段达到 UI 刷新的目的。示例使用更推荐的函数组件形式编写~

React Router

  React Router 是完整的 React 路由解决方案，保持 UI 与 URL 同步。拥有简单的 API 与强大的功能例如代码缓冲加载、动态路由匹配、以及建立正确的位置过渡处理的能力
